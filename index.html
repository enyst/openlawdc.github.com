openlawdc

A team of volunteers working to distribute, process, and analyze Washington DC's laws.

Turning laws constrained by expensive books and lackluster commercial portals into common knowledge and free content.

Want to help? Get access to the GitHub organization, talk on #openlawdc on freenode, and fix open issues in a project.

Read more at Freedom to Tinker, BoingBoing, DCist, Greater Greater Washington, Huff Post Local

/browser

a fast, open-source parser for the unofficial DC Code

/home-rule-act

an open-source, simple reader for the District of Columbia Home Rule Act

/dc-decoded

a parser for the unofficial released code into content for /browser and State Decoded
